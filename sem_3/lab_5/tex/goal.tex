\section{Цель работы}

Изучить особенности работы со строками в языке Java, освоить использование регулярных выражений для поиска, проверки и обработки текстовой информации.

\section{Введение}
Строки — один из наиболее часто используемых типов данных в программировании. В Java строки представлены неизменяемым классом \texttt{String}, что накладывает определённые особенности на их использование. Кроме того, для эффективной работы с текстом в Java предусмотрена мощная поддержка регулярных выражений через пакет \texttt{java.util.regex}. В данной лабораторной работе рассматриваются ключевые аспекты работы со строками и практическое применение регулярных выражений для решения типовых задач обработки текста.
